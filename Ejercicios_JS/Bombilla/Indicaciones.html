Ejercicio 1: Bombilla Interactiva

JavaScript puede modificar atributos del HTML. En este caso, cambiamos el atributo src de una imagen.

Instrucciones del Ejercicio

  1. Descarga la imagen de la bombilla encendida aquí
  2. Descarga la imagen de la bombilla apagada aquí
  3. Crea un botón para encender la luz con su respectivo id
  4. Crea una imagen y referencia la imagen pic_bulboff
  5. Crea un botón para apagar la luz con su respectivo id
  6. Declara 3 variables con los botones y la imagen usando document.getElementById()
  7. Agrega el evento click con addEventListener
  8. Crea una función Encender() que cambie el atributo src
  9. Crea una función Apagar() que cambie el atributo src
